Takeaways

  • GPUs and tools for doing ML/AI can be re-purposed to solve very specific astronomy problems like forward modeling of data, but you need to think about using traditional methods in the new tools.
  • Algorithms for data compression of videos can give us significant compression while maintaining astronomical data quality standards — but only if we adapt them to our needs instead of using them as they are.
  • Software and algorithms made for tech domains can often be more useful for astronomy if you use them in an unintended way.
